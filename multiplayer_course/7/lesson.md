# Урок 7. Результаты и перезапуск игры

Завершение игрового цикла так же важно, как и его начало. В этом уроке мы научимся подводить итоги матча и возвращать всех игроков в лобби для новой игры.

## 1. Экран результатов (Result Scene)

Когда условие победы достигнуто, игра не должна просто закрываться. Мы создаем отдельную сцену `ResultScene`, которая показывает:
- Кто победил.
- Финальный счет.
- Кнопку или клавишу для возврата в меню.

**Как передать данные между сценами?**
Самый простой способ — использовать глобальную переменную или объект `RESULT`, который заполняется в `GameScene` перед переключением.

## 2. Событие Game Over

Событие об окончании игры всегда должен рассылать **Хост**, как только он зафиксировал победу одного из игроков.

**Логика на Хосте (в GameScene.update):**
```python
if scores[winner_id] >= 10:
    # Рассылаем всем итоги
    ctx.send("game_over", {"winner": winner_id, "scores": scores})
    # Переходим на экран результатов локально
    s.scene.set_scene_by_name("result")
```

## 3. Синхронный перезапуск (Restart)

Обычно право нажать «Играть снова» есть только у Хоста. Когда он нажимает `R`, он отправляет событие `"restart"`, и все клиенты одновременно возвращаются в `MenuScene`.

**Пример кода (ResultScene.update):**
```python
if ctx.is_host and s.input.was_pressed(pygame.K_r):
    ctx.send("restart", {})
    s.scene.set_scene_by_name("menu")

# Чтение на клиенте
for msg in ctx.poll():
    if msg.get("event") == "restart":
        s.scene.set_scene_by_name("menu")
```
