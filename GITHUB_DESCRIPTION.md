**In English:**

SpritePro is a high-level 2D game framework for Python built on Pygame. Instead of wiring up rendering, camera, and input by hand, you get a ready-made stack: sprites auto-draw, the camera follows the player in one line, animations use a fluent API (`player.DoMove((100, 200), 1).SetEase(Ease.OutQuad)`), and UI plus layout (flex, grid, circle) work out of the box. It includes particles, PlayerPrefs-style saves, TCP multiplayer with EventBus, and a 10-lesson mini-course from basic messaging to a full multiplayer mini-game. The goal is to turn 2D game creation from low-level plumbing into something fast and enjoyable.
