## Инициализация и настройка

### Инициализация
```python
import spritePro

# Инициализация библиотеки
spritePro.init()

# Создание окна
spritePro.get_screen((800, 600), "Моя игра")
```

### Основные константы
- `spritePro.WH_CENTER`: Центр экрана
- `spritePro.screen`: Основная поверхность окна
- `spritePro.clock`: Объект для контроля FPS

## Основные классы

### Sprite
Базовый класс для всех спрайтов.

#### Свойства
- `size`: Размер спрайта (ширина, высота)
- `pos`: Позиция спрайта (x, y)
- `velocity`: Вектор скорости
- `speed`: Базовая скорость движения
- `angle`: Угол поворота
- `scale`: Масштаб
- `alpha`: Прозрачность (0-255)
- `state`: Текущее состояние
- `color`: Цветовой оттенок

#### Методы
- `set_image(image_source, size=None)`: Установка изображения
- `update(window)`: Обновление состояния
- `move(dx, dy)`: Перемещение
- `move_towards(target_pos, speed=None)`: Движение к цели
- `set_velocity(vx, vy)`: Установка скорости
- `rotate_to(angle)`: Поворот на угол
- `set_scale(scale)`: Установка масштаба
- `set_alpha(alpha)`: Установка прозрачности
- `set_color(color)`: Установка цвета
- `limit_movement(bounds)`: Ограничение движения

### GameSprite
Расширяет базовый Sprite для игровых объектов.

#### Дополнительные свойства
- `health`: Текущее здоровье
- `max_health`: Максимальное здоровье
- `is_alive`: Флаг жизни

#### Дополнительные методы
- `take_damage(amount)`: Получение урона
- `heal(amount)`: Восстановление здоровья
- `resurrect()`: Воскрешение
- `on_health_change(callback)`: Обработчик изменения здоровья
- `on_death(callback)`: Обработчик смерти

### PhysicSprite
Добавляет физическую симуляцию.

#### Дополнительные свойства
- `mass`: Масса
- `gravity`: Гравитация
- `bounce_enabled`: Включен ли отскок
- `is_grounded`: На земле ли объект
- `friction`: Трение

#### Дополнительные методы
- `apply_force(force)`: Применение силы
- `update_physics(delta_time)`: Обновление физики
- `bounce(normal)`: Обработка отскока

## Компоненты

### Text
Компонент для работы с текстом.

#### Свойства
- `text`: Текст для отображения
- `font_size`: Размер шрифта
- `color`: Цвет текста
- `font`: Шрифт

#### Методы
- `set_text(text)`: Установка текста
- `set_color(color)`: Установка цвета
- `set_font_size(size)`: Установка размера шрифта

### Button
Интерактивная кнопка.

#### Свойства
- `text`: Текст кнопки
- `pos`: Позиция
- `size`: Размер
- `is_hovered`: Наведена ли мышь
- `is_pressed`: Нажата ли кнопка

#### Методы
- `on_click(callback)`: Обработчик клика
- `on_hover(callback)`: Обработчик наведения
- `update()`: Обновление состояния

### Timer
Компонент для работы со временем.

#### Свойства
- `duration`: Длительность
- `is_running`: Работает ли таймер
- `progress`: Прогресс (0-1)
- `time_left`: Оставшееся время

#### Методы
- `start()`: Запуск таймера
- `pause()`: Пауза
- `resume()`: Возобновление
- `reset()`: Сброс
- `on_complete(callback)`: Обработчик завершения

### Health
Компонент для управления здоровьем.

#### Свойства
- `max_health`: Максимальное здоровье
- `current_health`: Текущее здоровье
- `is_alive`: Жив ли объект

#### Методы
- `take_damage(amount)`: Получение урона
- `heal(amount)`: Лечение
- `resurrect()`: Воскрешение
- `on_health_change(callback)`: Обработчик изменения здоровья
- `on_death(callback)`: Обработчик смерти

### MouseInteractor
Компонент для обработки взаимодействия с мышью.

#### Свойства
- `is_hovered`: Наведена ли мышь
- `is_pressed`: Нажата ли кнопка мыши
- `is_clicked`: Был ли клик

#### Методы
- `on_hover(callback)`: Обработчик наведения
- `on_press(callback)`: Обработчик нажатия
- `on_release(callback)`: Обработчик отпускания
- `on_click(callback)`: Обработчик клика
- `update()`: Обновление состояния

## Утилиты

### Surface
Функции для работы с поверхностями.

#### Функции
- `round_corners(surface, radius)`: Создание поверхности с закругленными углами
- `set_mask(surface, mask)`: Применение маски к поверхности
